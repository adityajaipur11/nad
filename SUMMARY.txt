═══════════════════════════════════════════════════════════════════════════════
                            CUSTOMIZATION GUIDE
═══════════════════════════════════════════════════════════════════════════════

Change Colors:
  Edit templates/base.html, find :root section, modify CSS variables:
  
  :root {
    --primary: #3b82f6;
    --success: #10b981;
    --warning: #f59e0b;
    --error: #ef4444;
  }

Change Refresh Rate:
  Find setInterval(loadStats, 3000) and change 3000 to desired milliseconds
  Examples: 5000 (5s), 10000 (10s), 30000 (30s)

Add Custom Styling:
  Edit <style> section in base.html or individual page templates

═══════════════════════════════════════════════════════════════════════════════
                            TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Issue: "Port 5000 already in use"
Fix:   Change port in app.py line 589: app.run(..., port=5001)

Issue: "No data showing on dashboard"
Fix:   1. Configure API key in Settings page
       2. Run a scan from Dashboard
       3. Check network tab in DevTools (F12)

Issue: "Buttons not responding"
Fix:   1. Clear browser cache (Ctrl+Shift+Del or Cmd+Shift+Delete)
       2. Hard refresh page (Ctrl+Shift+R or Cmd+Shift+R)
       3. Check browser console (F12) for errors

Issue: "Theme not saving"
Fix:   1. Check if localStorage is enabled
       2. Browser DevTools > Application > Local Storage
       3. Try incognito/private window

Issue: "API key error"
Fix:   1. Generate new key from https://openrouter.ai
       2. Copy full key (starts with sk-or-v1-)
       3. Test in Settings page
       4. Check for extra spaces/quotes
